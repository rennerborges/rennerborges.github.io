O que é a internet?

A internet é uma rede mundial que tem como objetivo interligar computadores para fornecer ao usuário o acesso a diversas informações.

ORIGEM

  • Nasceu na guerra fria
  • Projeto militar
  • Resposta ao Sputnik

ARPANET

  • Primeira rede de computadores
  • Criada pelo Departamento de Defesa dos EUA

EVOLUÇÃO

  • Começo como projeto militar
  • Liberação ás universidades
    • Possibilitou interligar os diferentes computadores
    • Permitiu desenvolver trabalhos cooperativos
  • Adoção de um protocolo aberto

INTERNET

  • O termo surgiu definitivamente em 1983
  • National Science Foundation criou uma ligação de alta velocidade
  • Governo americano decidiu financiar a formação de redes regionais

Internet como conhecemos
